# Student Grade Management System

This repository contains a simple **Student Grade Management System** implemented in Python using basic programming concepts, control flow, and data structures like lists and dictionaries.  

It allows users to manage student records efficiently, providing functionalities similar to real-world grade management systems while keeping the implementation beginner-friendly.

## Features
- Display all students along with their grades and average
- Add a new student with multiple grades
- Calculate average grade for a specific student
- Update grades for an existing student
- Interactive command-line interface

## How to Use
1. Clone or download this repository.
2. Open the `student_grade_system.py` file in your terminal or IDE.
3. Run the program using Python 3:
```bash
python student_grade_system.py
